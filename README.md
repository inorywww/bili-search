# bili-search

## 前言
该项目主要是爬取bilibili视频信息，然后通过前端输入查询数据，返回到后端，后端通过Elasticsearch获取结果，再把结果返回到前端渲染展示。来实现的本地一个搜索引擎。

## 功能概述
用户输入关键字，然后通过Elasticsearch在本地查找，最终将内容渲染到前端页面。

## 技术实现
- 前端：html+css+js，UI仿照bilibili搜索系统实现。
- 后端：python+flask
